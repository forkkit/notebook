## The Core Block Component

```jsx
import React from 'react'
import { ThemeProvider } from 'theme-ui'
import { Box } from 'theme-ui/layout'
import merge from 'lodash.merge'

export const Block = ({
  components,
  theme,
  styles,
  ...props
}) =>
  <ThemeProvider
    components={components}
    theme={merge({}, theme, {
      styles
    })}>
    <Box data-block {...props} />
  </ThemeProvider>
```

## Creating a Block

```jsx
import React from 'react'
import { Block } from 'theme-ui/blocks'

export default props =>
  <Block
    {...props}
    css={{
      // root element styles, i.e. this Block
    }}
    styles={{
      // child element styles
    }}
    theme={{
      // optional theme overrides
    }}
  />
```

## Notes

- Use child context to map one component to another, e.g. render `NavLink` in place of `a`
  - All styled UI components in scope at root, but sections can contextually change the rendered component
