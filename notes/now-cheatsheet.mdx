
# Zeit & Now Cheatsheet

## Add a redirect

1. Create an empty directory with a `now.json` config

  ```json
  {
    "version": 2,
    "alias": [
      "theme-ui.now.sh"
    ],
    "routes": [
      { "src": "/(.*)", "status": 301, "headers": { "Location": "https://theme-ui.com/$1" }}
    ]
  }
  ```

2. Use `now switch` to ensure it's set to the correct team
3. `now`
4. `now alias`
